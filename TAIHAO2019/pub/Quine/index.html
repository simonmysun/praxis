Quine

A quine is a computer program which takes no input and produces a copy of its own source code as its only output.

--from Wikipedia

Here is an interesting zip file. And here's a simpler one. I would be happy to introduce this "computer scientists' puzzle", but you have to first try to solve it yourself.

Write a quine and you will get the key.

There are two operations allowed here:

  • print N, prints the following N lines of input. the printed lines will not be executed.
  • repeat N M, repeat the last N lines of output, starting M lines from the end.

where N is a non-negative integer and M is a positive integer.

Write your code on the left input and the result will be shown on the right. If the result starts with "ERR" then it's an error message.

Once your submission satisfies the conditions, you will get the key for this level.
